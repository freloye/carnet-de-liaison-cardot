# Carnet de liaison Cardot

Interface smartphone destinée à la saisie quotidienne de Pierre-Yves.

## Lot 1

Cette première version contient :

- une page unique défilante ;
- une interface adaptée au smartphone ;
- les activités réalisées ;
- les tendances concernant Thérèse ;
- les indicateurs de Pierre-Yves ;
- une remarque facultative ;
- aucune transmission vers Apps Script à ce stade.

## Fichiers

- `index.html`
- `styles.css`
- `app.js`
- `config.js`

## Mise en ligne

Déposer les fichiers à la racine du dépôt GitHub, puis activer GitHub Pages depuis :

`Settings > Pages > Deploy from a branch > main / root`
